## 🎯 使用示例

### 基础播放
```bash
# 播放音频文件
./music-player song.wav

# 使用插件感知播放器
./music-player-plugin song.flac -p ./plugins
```

### 高质量转换
```bash
# 最佳质量播放
./music-player-enhanced song.wav --quality best --mode adaptive

# 转换到高采样率
./test-universal-converter input.wav output.wav --input-rate 44100 --output-rate 192000
```

### 插件管理
```bash
# 列出支持的格式
./music-player-plugin -l

# 显示插件信息
./music-player-plugin -i
```
